# Clearly Prepare Screen Addon



**GitHub:** [kabzon93region](https://github.com/kabzon93region)

**Клиентский мод для SPT 4.** Аддон к ClearlyPrepareScreen — UI матчмейкера. Дополнительные элементы интерфейса для экрана подготовки к рейду.



## Требования



- **ClearlyPrepareScreen** — оригинальный мод

- **SPT**: 4.0.x

- **BepInEx**: 5.4.x



## Установка



1. Скопировать ClearlyPrepareScreenAddon.dll в BepInEx/plugins/



## Совместимость



- client_only — только на клиенте, не требует серверной части
